# ITECH-LOAD-PC-CONTROL
Using python to control ITECH Electronic Load (using Serial)

## Usage

Change ITECH Electronic Load COMMUNICATE PROTOCOL to FRAME, BAUD RATE To 115200

## Requirements

        Python3
        pyqt5
        pyserial


## Python & PIP Install (Chinese)
